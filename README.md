#### Gestion des media queries

**Desktop**
```css
@media screen and (min-width: 1024px) and (max-width: 2000px)
```

**Tablette**
```css
@media (min-width: 500px) and (max-width: 1023px)
``` 

**Smartphone** 
```css
@media screen and (min-width: 0px) and (max-width: 500px)
```
